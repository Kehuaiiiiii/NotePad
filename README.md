# NotePad
南京邮电大学JAVA大作业------文本编辑器

①基本要求：设计类似于Windows Notepad（记事本）文本编辑器的程序，可以创建，编辑和保存文本格式的文件。
<br/>
②提高要求：在文本编辑时实现复制/粘贴功能；在文本中搜索/替换字符（字符串）；编辑Java源程序时语法关键字加亮；可打开二进制格式的文件，对其进行二进制形式的编辑并将编辑结果保存回文件中；在文本中以字符串形式插入当前的日期/时间。
